## ปลั้กอินช่วยออกรายงาน PageSpeed Insights และยอดขายประจำเดือน พร้อมเก็บ Snapshot ##

เคยเบื่อไหม ? ที่ต้องทำรายงานประสิทธิภาพและยอดขายของเว็บไซต์ ใน Microsoft Word หรือ Google Slider

### ปลั้กอินนี้อาจช่วยคุณได้ ! ###

ปลั้กอินนี้จะดึงยอดขาย (Net Sales) จาก WooCommerce ของเดือนปัจจุบันมาอัตโนมัติ ส่วนของ PageSpeed Insights ทางปลั้กอินมีช่องกรอกพร้อมสำหรับ Performance ของทั้ง Desktop และ Mobile

เมื่อกดออกรายงาน ระบบจะบันทึก Snapshot ลงในระบบโดยอัตโนมัติ ซึ่งสามารถเข้ามาดูข้อมูลย้อนหลังเมื่อไหร่ก็ได้

## วิธีติดตั้งปลั้กอิน ##
ไปที่ <code>/wp-admin/plugin-install.php</code> และอัปโหลดไฟล์ zip ของปลั้กอิน จากนั้นสามารถ <b>Activate</b> แล้วเริ่มใช้งานปลั้กอินจากเมนูได้เลย !
